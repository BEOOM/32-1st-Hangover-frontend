# Hangover

## 🍻프로젝트 Front-end/Back-end 소개

`VIVINO` 사이트 클론코딩 프로젝트  
기존의 와인 추천 사이트를 `hangover`라는 단어에 맞추어, 리뉴얼 하였습니다.
기존 와인 목록을 전체 주류로 변경, 도수와 숙취도를 기반으로하는 주류 추천 및 소개 페이지입니다.
